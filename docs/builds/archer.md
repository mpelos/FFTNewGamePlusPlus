# Archer Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a Perseus Bow Archer with Aim, Arts of War secondary, Speed Surge, Attack Boost, and
Movement +3. The build uses long range both for damage and for equipment/stat disruption.

## Ability Package

```text
Primary: Aim
Secondary: Arts of War
Reaction: Speed Surge
Support: Attack Boost
Movement: Movement +3
```

## NG++ Enemy Adaptation

Archer is best as lane pressure. Arts of War makes ranged break pressure dangerous, so use it when the
map is already about gear or route denial. If the battle needs simpler chip, use Item or no secondary
and focus on height, bow range, Reflexes/Speed Surge, and mobility.
