# Arithmetician Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends Arithmetician with Arithmeticks, Iaido secondary, Shirahadori, Magick Boost, and
Teleport, while explicitly recommending that Arithmeticks often works better as a secondary on a faster
or higher-MA job.

## Ability Package

```text
Primary: Arithmeticks
Secondary: Iaido
Reaction: Shirahadori
Support: Magick Boost
Movement: Teleport
```

## NG++ Enemy Adaptation

Arithmetician is campaign-breaking if used freely. For enemy design, prefer giving Arithmeticks to a
specific caster only when the battle is meant to be a readable map-wide math puzzle. Otherwise use the
job's concept as inspiration and avoid unrestricted CT/level/height nuking.
