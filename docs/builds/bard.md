# Bard Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a Bard support build with Bardsong, Items secondary, Mana Shield, Throw Items, and
Manafont. The build is a global support engine rather than a direct attacker.

## Ability Package

```text
Primary: Bardsong
Secondary: Items
Reaction: Mana Shield
Support: Throw Items
Movement: Manafont
```

## NG++ Enemy Adaptation

Bard is a global-scaling support unit. It is strongest when protected by a front line, but it can make
long fights snowball. For enemy use, pick songs that support the encounter's main pressure and avoid
turning the fight into passive cleanup.
