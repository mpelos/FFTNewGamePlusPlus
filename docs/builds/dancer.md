# Dancer Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a Dance support build with Dance, Items secondary, Mana Shield, Throw Items, and
Manafont. The build uses global dance pressure while keeping emergency item support available.

## Ability Package

```text
Primary: Dance
Secondary: Items
Reaction: Mana Shield
Support: Throw Items
Movement: Manafont
```

## NG++ Enemy Adaptation

Dancer creates global pressure, so constrain the exact Dance list per battle. Use HP/MP chip when you
want a soft clock; avoid stacking several Dancers or hard-control dances unless the encounter is built
around that single performer engine.
