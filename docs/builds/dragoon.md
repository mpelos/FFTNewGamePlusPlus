# Dragoon Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a physical Dragoon with Jump, Throw secondary, Shirahadori, Attack Boost, and
Teleport. The guide highlights Jump damage and uses Throw as instant ranged fallback; community
discussion also mentions Ninja + Jump because high Speed helps Jump land before targets move.

## Ability Package

```text
Primary: Jump
Secondary: Throw
Reaction: Shirahadori
Support: Attack Boost
Movement: Teleport
```

## NG++ Enemy Adaptation

Dragoon works best as visible panel pressure. Haste, speed gear, or high mobility make Jump more
urgent; Immobilize or Leg Shot support makes Jump more reliable. Keep the landing-marker counterplay
visible unless the battle is meant to be a spike.
