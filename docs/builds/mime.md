# Mime Build Notes

Source basis: internet research only.

## Best Reference Build

Mime is an exception: the job has no normal ability slots, no equipment, and depends on copying allied
actions. Online job discussions therefore treat Mime less as a standalone build and more as a team
engine.

## Ability Package

```text
Primary: Mime behavior
Secondary: none
Reaction: none
Support: none
Movement: none
```

## NG++ Enemy Adaptation

Avoid Mime unless the battle is explicitly designed around a visible mimic puzzle. It is hard to make
readable because its threat depends on surrounding units rather than its own kit. If used, place it
where copied actions are obvious and answerable.
