# Monk Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a versatile attacker Monk using Martial Arts, Steal secondary, Mana Shield, Attack
Boost, and Manafont. Community discussion also strongly values Monk with Dual Wield for extreme
unarmed damage.

## Ability Package

```text
Primary: Martial Arts
Secondary: Steal, Fundaments, or Items
Reaction: Mana Shield or Counter
Support: Attack Boost or Dual Wield
Movement: Manafont or Movement +3
```

## NG++ Enemy Adaptation

Monk is excellent as a bruiser because it brings damage, ranged pressure, Chakra, Revive, and status
cleanup without equipment dependence. For enemies, Dual Wield Monk is a spike; Attack Boost Monk is
usually easier to tune.
