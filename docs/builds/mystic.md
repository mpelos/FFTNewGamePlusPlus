# Mystic Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends an Arithmeticks-focused Mystic with Mystic Arts, Arithmeticks secondary, Shirahadori,
Magick Boost, and Teleport. A pole variant exists for fallback melee when casting is poor.

## Ability Package

```text
Primary: Mystic Arts
Secondary: Arithmeticks
Reaction: Shirahadori
Support: Magick Boost
Movement: Teleport
```

## NG++ Enemy Adaptation

Mystic is a soft-control caster. For enemy design, it should weaken or shape decisions without hard
turn deletion. Avoid stacking too much Stop/Disable-style pressure; use Faith manipulation, debuffs,
and defensive support to amplify the battle's main engine.
