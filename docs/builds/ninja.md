# Ninja Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends a max-damage brawler Ninja with Throw, Martial Arts secondary, Mana Shield, Brawler,
and Manafont. Community discussion also highlights Ninja + Jump because high Speed helps Jump land
before targets move.

## Ability Package

```text
Primary: Throw
Secondary: Martial Arts or Jump
Reaction: Mana Shield or Vanish
Support: Brawler
Movement: Manafont or Movement +3
```

## NG++ Enemy Adaptation

Ninja is naturally lethal due to Speed and innate Dual Wield. Use it sparingly as a flanker, backline
punisher, or item-thrower. Martial Arts turns it into a barehanded blender; Jump turns it into a fast
execution threat.
