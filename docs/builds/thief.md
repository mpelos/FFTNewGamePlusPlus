# Thief Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 presents Thief as an early fast offensive build with Steal, Martial Arts secondary, Auto-Potion,
Brawler, and Movement +2. The guide notes that Thief itself is not usually the strongest main battle
job, but speed and stealing remain valuable.

## Ability Package

```text
Primary: Steal
Secondary: Martial Arts
Reaction: Auto-Potion
Support: Brawler
Movement: Movement +2
```

## NG++ Enemy Adaptation

Thief is best as a pressure/loot-disruption unit, not a raw killer. Use high Speed, Steal, and mobility
to force the player to respect positioning. Pair with disabling allies if you want theft to be a real
threat.
