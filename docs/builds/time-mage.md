# Time Mage Build Notes

Source basis: internet research only.

## Best Reference Build

Game8 recommends an Arithmeticks-focused Time Mage with Time Magick, Arithmeticks secondary,
Shirahadori, Magick Boost, and Teleport. The guide notes that Time Mage keeps Meteor as an option when
Arithmeticks cannot target the desired enemies.

## Ability Package

```text
Primary: Time Magick
Secondary: Arithmeticks or Black Magick
Reaction: Shirahadori
Support: Magick Boost
Movement: Teleport
```

## NG++ Enemy Adaptation

Time Mage is strongest as tempo glue. Haste, Slow, Float, Mana Shield, and Teleport all make a team
more coherent. Stop or other hard-lock patterns should be used only when the battle explicitly wants a
controlled status puzzle.
